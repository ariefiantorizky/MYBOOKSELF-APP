About MyBookSelf

MyBookSelf adalah sebuah aplikasi berbasis website yang dirancang untuk semua pengguna khususnya para pembaca dan senang mengoleksi buku. Website ini dikembangkan menggunakan teknologi Javascript dan Untuk penyimpanan berbasis LocalStorage, yang artinya setiap browser akan berbeda - beda penyimpanannya.

Developer ingin mengembangkan website ini yang nantinya akan bermanfaat bagi semua pengguna. Selain itu akan ada update secara berkala agar pengguna merasa puas terhadap aplikasi website ini. Jika ada kritik dan saran, silahkan kirimkan melalui No. +62xxxxxxxxxx
